🍎Vocab Fruit Slasher
An interactive English vocabulary learning game built with HTML5 Canvas. Practice your listening and word recognition skills by "slashing" the correct fruits based on real-time voice prompts.

✨ Key Features
Voice-Guided Learning: Integrated Web Speech API (TTS) that reads the target word aloud continuously to reinforce auditory memory.

Physics-Based Gameplay: Fruits follow natural parabolic trajectories with gravity effects, providing a smooth and engaging experience.

Explosion Effects: Satisfying particle explosion animations when a correct word is hit.

Smart Synchronization: Real-time detection ensures the target word is always one of the fruits currently visible on the screen.

Cross-Platform: Optimized for both Desktop (Mouse) and Mobile (Touch) interactions.

This is a pure front-end project. No installation or local server is required.

Simply double-click index.html to open it in your browser (Edge, Chrome, or Safari recommended).

Note: Due to browser autoplay policies, you must click the "Start Game" button to activate the audio features.

🛠️ Tech Stack
HTML5 Canvas: For high-performance game rendering and animations.

JavaScript (ES6+): Core game logic, physics engine, and particle systems.

Web Speech API: For high-quality text-to-speech synthesis.

💡 How to Play
Look at the Target Word in the top-left corner and listen to the pronunciation.

Find the fruit with the matching word as it flies across the screen.

Tap or click the fruit to "slash" it before it falls back down.

If a target word falls off-screen, the game automatically picks a new word from the remaining fruits.

Try to get the highest score possible within 60 seconds!
