# vocab-fruit-slasher
English Vocab Fruit Slasher 🍎🎮 An interactive educational game built with HTML5 Canvas. Practice English vocabulary by "slashing" the correct fruits based on voice prompts.  Features: ✅ Real-time Text-to-Speech (TTS) ✅ Dynamic physics &amp; explosion effects ✅ Mobile-friendly touch support ✅ 60s fast-paced challenges  Perfect for English learners!
